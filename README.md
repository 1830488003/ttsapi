# 硅基流动语音扩展

这是一个适用于 **SillyTavern** 的第三方 TTS 扩展，用来为聊天楼层添加语音朗读能力。安装并配置后，你可以直接点击消息旁边的耳机按钮，让插件调用 **硅基流动 TTS API** 生成语音并在前端播放。

## 功能简介

- 为聊天消息注入耳机朗读按钮
- 支持“全文模式 / 对话模式”两种朗读方式
- 支持选择不同模型与默认音色
- 支持设置播放语速、音调、用户性别
- 支持仅在角色消息上显示按钮
- 内置运行状态与调试日志，方便排查问题
- 支持检查扩展更新

## 使用流程

1. 打开扩展设置页
2. 填入可用的 **硅基流动 API Key** 并保存
3. 选择要使用的模型
4. 选择默认音色
5. 按需设置朗读模式、语速、音调等参数
6. 进入聊天页面，点击消息旁边的耳机按钮开始朗读

## 模式说明

### 全文模式

适合直接朗读整段消息内容，适用于旁白、叙述、长段文本等场景。

### 对话模式

适合对白类文本。对于部分模型，插件会先对文本进行整理后再生成语音，因此在内容较长或结构较复杂时，生成时间会更久一些。

## 注意事项

1. 必须填写有效的硅基流动 API Key，否则无法生成语音
2. 长文本、复杂对白或特定模型下，生成速度可能较慢
3. 不同模型支持的音色、效果和表现可能存在差异
4. “播放语速”影响的是前端播放速度，不是模型生成速度
5. 手机端耳机按钮有时会收纳在消息右侧的更多菜单中

## 适用场景

- 角色对白试听
- 旁白/剧情段落朗读
- 移动端轻量语音播放
- 调试不同音色与模型效果
