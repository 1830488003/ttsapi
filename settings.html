新手使用教程
📖 点我展开硅基流动语音使用教程（推荐先看）

1️⃣ 先做什么？

先准备一个硅基流动 API Key，并填到下面的输入框里保存。

2️⃣ 模型怎么选？

- 全文模式：适合整段叙述直接朗读

- 对话模式：适合只读对白，MOSS 模型会先走 DeepSeek 整理脚本

- fnlp/MOSS-TTSD-v0.5 更适合多说话人对话

3️⃣ 音色和语速怎么调？

① 选择你喜欢的默认音色

② 调整“播放语速”，这是浏览器播放速度，不是模型生成速度

③ 需要对话人物判断更准时，可以设置“用户性别”

4️⃣ 怎么使用？

① 进入聊天界面

② 电脑端可以直接点每层楼旁边的耳机按钮；手机端耳机按钮通常会隐藏在消息右侧的“三个点”按钮菜单里

③ 点击后插件会自动生成语音并播放

④ 如果你使用的是 对话模式 + MOSS，因为会先调用 DeepSeek 做情绪分析和对白整理，生成时间通常大约要十几秒，这期间请不要重复点击按钮

💡 推荐新手最简单用法： 填 Key → 保存 → 选模型 → 选音色 → 点楼层按钮直接听。
